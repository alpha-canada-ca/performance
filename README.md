## Setup

* Update */php/config.php* with the values of your API
* Include a */keys/* folder with a *secret.pem*.

## What is needed

* PHP 7.2 and above
  * If that cannot be provided, above PHP 5.5.
* PHP with the extensions enabled:
  * curl;
  * openssl; and
  * mongodb
* OpenSSL above 1.0.1 (or latest)
* MongoDB
